# Multi-Vendor Marketplace App 🚀  

This project is a mobile-based multi-vendor platform built with Flutter, designed to bring sellers and buyers together in a seamless and intuitive marketplace experience.



## 🌟 Key Features  
- Seller Dashboard:  
  Sellers can effortlessly list, update, and manage their products.  
- **Product Discovery**:  
  Users can explore and search for products with advanced filtering and categorization.  
- **Order Management**:  
  Place orders with ease and track their status in real-time.  
- **Notifications**:  
  Stay updated with push notifications for key activities like order updates, promotions, and more.



## ⚙️ Backend Integration  
The app integrates with existing backend microservices to power its core functionalities:  
- **Authentication**: Secure login and user management.  
- **Product Management**: Efficient handling of product listings, categories, and inventory.  
- **Order Processing**: Robust system for creating, updating, and managing orders.  
- **Notifications**: Real-time updates to enhance user engagement.



## 🌐 Technologies Used  
- **Flutter**: For building a cross-platform mobile experience.  
- **REST APIs**: Communication with backend microservices.  
- **Push Notifications**: To keep users informed and engaged.  
- **Microservices Architecture**: Ensures scalability and maintainability.



## 🚀 Getting Started  
### Prerequisites  
- Flutter installed on your machine.  
- Access to the backend microservices for authentication, product management, and orders.  

### Installation  
1. Clone the repository:  
   ```bash
   git clone https://github.com/AdikaNathaniel/footwear_client_mobile-app.git
   ```  
2. Navigate to the project directory:  
   ```bash
   cd MultiVendorMarketplace
   ```  
3. Install dependencies:  
   ```bash
   flutter pub get
   ```  
4. Run the app:  
   ```bash
   flutter run
   ```
